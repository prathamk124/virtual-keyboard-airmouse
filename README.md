# Virtual Keyboard & Air Mouse

An AI-powered touchless computer interaction system built using Python, OpenCV, and MediaPipe.

## Features

- Hand Tracking
- Air Mouse
- Virtual Keyboard
- Pinch Gesture Detection
- Smooth Cursor Control
- OS-Level Mouse and Keyboard Interaction

## Tech Stack

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pynput
